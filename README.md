# Proyecto de Autenticación Centralizada (LDAP + Kerberos)
Este sistema permite la autenticación segura de usuarios y gestión de perfiles con fotos personalizadas.

## Estructura
- **src/**: Código fuente del sitio web.
- **scripts/**: Scripts de administración y configuración.
- **config/**: Archivos LDIF de respaldo.

## Instalación (Para el administrador)
1. Clonar el repositorio.
2. Ejecutar el despliegue automático:
   ```bash
   chmod +x deploy.sh
   ./deploy.sh
